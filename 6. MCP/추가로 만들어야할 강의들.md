# MCP(Model Context Protocol) 실무 마스터 커리큘럼

- FastMCP

## 1단계: MCP의 이해와 환경 구축 (기초)
- MCP의 개념과 등장 배경: Anthropic이 제안한 개방형 표준의 의미.
- 아키텍처 이해: 호스트(Host), 클라이언트(Client), 서버(Server)의 관계.
- 환경 설정: Claude Desktop 설정 및 기본 MCP Server 연결 실습.
- Inspector 활용: MCP Inspector를 사용한 서버 디버깅 및 메시지 흐름 파악.

## 2단계: MCP Server 개발 입문 (SDK 활용)
- 사용할 라이브러리: FastMCP
- 핵심 컴포넌트 구현:
  - Resources: 데이터 읽기 전용 엔드포인트 만들기.
  - Tools: LLM이 실행할 수 있는 함수(Function) 정의하기.
  - Prompts: 재사용 가능한 프롬프트 템플릿 설계.
- 로컬 서버 배포: stdio 기반의 로컬 연결 및 테스트.

## 3단계: 외부 데이터 및 API 연동 (심화)
- 사용할 라이브러리: FastMCP, langchain-mcp-adapters, httpx, sse-starlette
- 데이터베이스 연결: SQLite, PostgreSQL 등을 MCP로 연결하여 데이터 질의하기.
- API 통합: GitHub, Google Search, Slack 등 외부 서비스 API 연동 실습.
- 보안 및 인증: API Key 관리 및 사용자 권한 제어 기초.
- 파일 시스템 제어: 안전한 파일 읽기/쓰기 권한 설정.
- 4단계: 멀티 에이전트 및 RAG 확장 (응용)
- Agentic Workflow: LangGraph나 다른 프레임워크와 MCP 연결 전략.
- RAG + MCP: 지식 베이스(Knowledge Base)를 MCP Resource로 추상화하여 답변 성능 높이기.
- 커스텀 도구 최적화: LLM이 도구를 더 정확하게 선택하게 만드는 Parameter 설계 기법.

## 5단계: 실전 프로젝트 및 배포 (실습)
- 실전 프로젝트 (택 1):
- 개인 비서형 MCP (캘린더 + 메일 + 할 일 관리)
- 개발 생산성 MCP (코드 리뷰 + 이슈 트래킹 + 문서 자동화)
- 데이터 분석 MCP (DB 쿼리 + 시각화 도구 연결)
- 엔터프라이즈 고려사항: 원격 MCP 서버 구성 및 확장성 확보 방안.
