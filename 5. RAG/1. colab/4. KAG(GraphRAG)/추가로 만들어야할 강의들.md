# LangGraph + Neo4j 기반 GraphRAG & HybridRAG 강의 커리큘럼

## 1. 오리엔테이션 & 전체 아키텍처 개요
- RAG 기술 진화: Vanilla RAG → GraphRAG → HybridRAG
- GraphDB와 VectorDB의 역할
- Neo4j로 관계 기반 추론 구현
- LangGraph로 Pipeline/Agent Workflow 관리
- 실습 환경 구성  
  (LangGraph, Neo4j, PGVector/Chroma)

## 2. LangGraph 기본기
### (1) LangChain vs LangGraph 차이
- LangGraph: 상태 머신 기반 LLM 파이프라인
- Graph 컴파일 개념
- Node / Edge / State 구조 이해

### (2) LangGraph 실습
- 간단한 파이프라인 구현
- 노드 간 데이터 전달 방식
- 조건 분기, Retry, Tool 호출

## 3. Neo4j 기초 + Graph Embedding
### (1) Neo4j 기초
- Label / Node / Relationship
- Cypher 기초 문법
- GDS(Graph Data Science) 소개

### (2) Graph Embedding
- Node Embedding vs Text Embedding
- FastRP, Node2Vec 개념 및 실습
- Neo4j에 Embedding 저장

## 4. GraphRAG 아키텍처
### (1) GraphRAG 필요성
- 문서 기반 RAG 한계
- 관계 기반 추론의 장점

### (2) GraphRAG Pipeline 구조
1. Query Understanding  
2. Entity Linking  
3. Graph Query(Cypher)  
4. Context Ranking  
5. Context Assembly  
6. LLM 답변 생성

### (3) 실습
- PDF → Entity 추출 → Neo4j 저장
- Graph Retriever 구현
- LangGraph 기반 GraphRAG 파이프라인 만들기

## 5. VectorDB 기반 Semantic RAG 복습
- Embedding 기반 의미 검색 구조
- PGVector / Chroma / Elasticsearch 비교
- Cosine / Dot / L2 / HNSW 개념
- Hybrid Query(BM25 + Vector Search)

**실습:** 벡터 저장, Retriever 구현, LangGraph + VectorDB 연동

## 6. Graph + VectorDB Hybrid RAG
### 목적
- Graph: 구조 기반 정밀 추론
- Vector: 의미 기반 고리콜 검색

### 두 가지 Hybrid 패턴
1. Graph First → Vector Secondary  
2. Vector First → Graph Expansion
